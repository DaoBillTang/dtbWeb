Django学习：
	python -m venv [name] #创建新的虚拟环境
	cd [虚拟环境目录]\ Scripts  -> activete.bat	#启动虚拟环境
	cd [虚拟环境目录]\ Scripts(也可以全路径直接启用)  -> django-admin startproject [项目名称]
	python manager.py runserver [可以跟端口号，默认8000]
	python manager.py startapp [app名字]
